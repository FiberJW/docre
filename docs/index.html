/** # Example

Top-level docs right here.

you end it with a semicolon, that's how you know it's serious.

A block in the top-level thing will be evaluated at the end of the file.

```
let x = 10;
let z = awesome();

```

## Functions

You can use the `@doc` pragma to include the docs for individual items.

So this `@doc awesome, Something, party`, becomes:

@doc awesome, Something, party

## Just dumping it all in

An `@all` annotation will dump all (exported) values with their associated documentation.

@all
